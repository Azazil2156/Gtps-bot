# GTPS Discord Bot

Bot Discord untuk berinteraksi dengan API GTPS (GrowTopia Private Server) menggunakan API GrowSoft.

## Fitur

- ✅ Info player GTPS (`!player <username>`)
- ✅ Status server GTPS (`!status`)
- ✅ Help command (`!help`)
